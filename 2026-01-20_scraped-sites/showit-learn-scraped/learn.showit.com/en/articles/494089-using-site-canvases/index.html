Using Site Canvases

Share site canvases across multiple pages for quick editing

Written by Jason Lackey
Updated over 9 months ago
Table of contents

A site canvas is a special universal type of canvas in Showit that you can make site-wide changes to quickly and easily.

Site canvases are best used for repeating sections across multiple pages like:

  • Mobile Navigation

  • Menus & Headers

  • Social Links

  • Footers

How Site Canvases Work

Site Canvases are great for making updates across your site quickly. However some settings will act

Universal Settings

The design elements on the Site Canvas (images, text, graphics) and Click Actions WILL update across all instances of the Site Canvas.

Site Canvas Default Settings

The Canvas tab in the right side Properties Panel (ie. stacking order, sticky) are the default settings for a site canvas. This means that if this site canvas is already in use on some pages, it WILL NOT update those instances.

These settings are the defaults that will be used whenever that Site Canvas is first added to a page. Once it's on a page, those settings in the right sidebar are specific to how the canvas will act on that page.

Adjusting Site Canvas Settings

Site canvases have two layers of settings. One is the Page Specific Layer that you can use to update settings that will be unique to that page, and the other is the Site Canvas Layer, where you can set universal and default settings for the Site Canvas.

The Page Specific Layer

The first layer of settings is the page specific settings that will not update across every page. You can access and adjust these settings by clicking on the Site Canvas without clicking the blue "Edit" button.

You will know you're editing the Page Specific Layer because you'll still be able to see your full page design and the stripes on this Site Canvas there. Hidden to Start, Sticky, and Stacking Order are the main settings located here that will not update across the site when changed.

The Site Canvas Layer

The second layer of settings is found after clicking the blue "Edit" button. You'll know you're editing the Site Canvas Layer when you are viewing the site canvas alone, and the stripes are gone.

Here you can make changes to Universal Settings for your site canvas, and to the Site Canvas Defaults Settings, as mentioned above.

Site Canvas Sets

You can quickly create sets of Site Canvases and add them to your pages with Site Canvas Sets.

Site Canvas Sets can be found and created just below your Site Canvas list in the left side Site Panel.

You can find out how to set up and use Site Canvas Sets in this article here.
