Site Canvas Sets

Group site canvases together to quickly add them to your pages in Showit

Written by Nicholas Sinden
Updated over 9 months ago
Table of contents

Adding multiple site canvases to your pages has never been easier! A Site Canvas Set is a standardized set of Site Canvases that you can apply across select pages or your entire site with the click of a button.

Creating a Site Canvas Set

To create a Canvas Set, make sure you have clicked on the Site Tab in the left side Site Panel. Then, at the very bottom of the Site Panel, you'll see an option called Site Canvas Sets with a + icon. Depending on how many pages are in your design, you may need to scroll down to find this section.

Name Your Site Canvas Set

The new Site Canvas Set that you have created can be found at the bottom of your list of Canvas Sets.

You can then rename the Site Canvas Set there, or in the right side Properties Panel under Set Properties.

Add Site Canvases to Your Site Canvas Set

You can add Site Canvases and make changes to your Canvas Set by clicking the Canvas Set tab in the left side Site Panel.

From there, you will see that your Canvas Set has a Top and Bottom. This is so that any Site Canvases that are added to the Top will always be found at the top of your Page, and any Site Canvases added to the Bottom will always be found at the bottom of your Page.

Note: The Top tab is most commonly used for Navigation Menus, Mobile Menus, Banners, etc. The Bottom tab is most commonly used for Footers.

You can add Site Canvases that you have already created to the Top and Bottom of your Canvas Set by clicking on the + to the right of the Top or Bottom tab and selecting an option from the drop down menu.

You can create multiple Canvas Sets in this way, so that you can have different sets of Site Canvases for different types of pages!

Note: Setting a Canvas Set as Default will make is that this Canvas Set is applied to every new page that is added.

Applying a Site Canvas Set to Your Pages

You can apply a Canvas Set to a multiple Pages with the click of a button. To do this feature, choose one of the Canvas Sets, and then choose from a few options under the Set Properties tab on the right. You can choose to add this Canvas Set to all pages, or you can select which individual pages you would like to add it to.

Then hit the blue Save button at the bottom.

Note: If you are setting up Site Canvas Sets for the first time you may have Site Canvases already on your Pages. You can delete these duplicates from your Pages while keeping your set.

After you have done this, choose a Page that the Canvas Set has been applied to. You will see the Canvas Set at the top and bottom of your Canvases.

You can also add a Canvas Set to your Page by choosing one from the Page Info/Template Info tab in the right side Properties Panel.

Delete Any Duplicate Site Canvases

If you already have Site Canvases set on your page, you may have duplicates after adding a Canvas Set for the first time. Make sure to delete any duplicate Site Canvases.

Note: You can still add and use Site Canvases outside of Site Canvas Sets. These you will add individually to your pages.
