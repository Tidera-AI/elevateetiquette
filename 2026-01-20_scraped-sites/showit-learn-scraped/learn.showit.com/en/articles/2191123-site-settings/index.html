Site Settings

The site settings section in the Showit platform and what functions you can access there

Written by Jeff Willems
Updated over 6 months ago
Table of contents

You're probably already familiar with the Design Settings in the Site Tab in the left side Site Panel inside of Showit. Now let's talk about the Site Settings found in that same section.

Access Your Site Settings

First make sure you have the Site Tab selected in the left side Site Panel, then click Site Settings.

Below we'll take a look at the different sections that make up your Site Settings, as well as what you can do there.

The Site Information Tab

Here you can adjust your Favicon, your Site Name, your Site Languages, your Custom Domain and your Vanity URL Slug. You also have links to your Publish, Vanity, and Preview URLs.

Favicon

This is where you can set the favicon, or the small image the goes next to your page name in a browser tab for your website.

​Site Name

This setting will change the name or your design that appears in the top left corner of your Showit Design App.

This setting is only used in the Design App, and is not where you would set your Site Name for SEO purposes. See here for more information on how Search Engines generate a Site Name.

​Site Language

Here you can set your site's default language tag with the HTML language attribute.

​Custom Domain

Here is where you can assign your website's Domain Name to one of your site designs. This will be the URL where you can find your live website. You can only have one Domain Name per Showit account.

If your domain is not yet available, see how to get that set up here.

⚠️ Submitting a domain set up request will effectively point your domain to Showit and away from whatever might currently be at your domain. Be sure you are ready to launch or have a landing page ready before putting in that request if you have a site currently at your domain.

Vanity URL Slug

Your Vanity URL is a tool that can be used for previewing your site before it is live. You can customize your Vanity URL if you wish, but it isn't necessary.

Publish URL

This is the URL at which your live Showit site will appear. If you have a domain set under Custom Domain, then your Publish URL will match that.

Preview URL

You can use this URL to view a preview version of your live site.

The Blog Tab

This Tab is for Showit users with a Basic or Advanced blog subscription.

If you do not have a Showit connected blog AND you have a Basic or Advanced blog subscription - You ca use this tab to submit a blog set up request or to request a migration (Migrations are only available on the Advanced blog subscription).

If you have a Showit connected blog - This is where you would set your Blog Domain which ensures Showit & WordPress have the necessary connection to communicate. If you are having blog trouble but your Blog Domain is set, check out these blog troubleshooting tips.

The Social Tab

Here you can set the URLs for any social links that you have set within your site.

The Integrations Tab

The Integrations Tab is a section of your Site Settings where you can globally add third party integrations to your Showit site. There are a few ready-made options for easy integration, including:

  • Google Analytics

  • Google Tag Manager

  • Meta Pixel

  • BDOW

You can also add Global Custom Code to your site, if the integration that you are looking to add to your site isn't already listed and needs to be added to all of your pages.
